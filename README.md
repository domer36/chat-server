# Chat Server
It's a server chat for use it click [here](https://domer36-chat.netlify.com)

# Instalation
Clone this repo

Create .env file and set the variable

``` PORT=3000 ```

``` FRONTENDPOINT=http://localhost:4200 ```

Enter at dir _server_chat_ and install dependencies.

``` cd server_chat ```

``` npm i ```

``` npm run dev ```
